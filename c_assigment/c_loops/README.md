Context- This folder of the repo consist of the chapter " Iteration statements/Loops". In the previous chapter we mainly dealt with selection statements which lets the program choose between 
different calculation bodies if certain conditions/expressions are satisfied. In this section, we move further into statements and introduce iteration statements or commonly known as loops.
Loops are typue of statements that lets a program run a loop body until a condition/expression is satisfied. In this section, we shall engage with three types of iteration statements. 
1. The while loop.
2. The do loop.  
3. The for loop.
